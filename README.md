# AI-Powered Industrial Water Optimization System

## Overview
This project is an AI-powered industrial water flow optimization dashboard developed using Java Swing, Ford-Fulkerson Algorithm, and Machine Learning.

The system simulates industrial water distribution inside a manufacturing plant and helps optimize water flow between processing units.

It includes:
- Maximum Flow Optimization
- Augmenting Path Visualization
- Bottleneck Detection
- Pipeline Utilization Analysis
- What-If Simulation
- AI-Based Water Demand Prediction

---

# Features

## Optimization Features
- Ford-Fulkerson Maximum Flow Algorithm

- Bottleneck Pipeline Detection
- Pipeline Utilization Monitoring
